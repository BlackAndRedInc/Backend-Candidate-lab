## Running the Environment

**Spinup MongoDB**

``` mongod ```


**Open a Mongo shell and switch to the DB** (in new terminal)

``` mongo ```
``` use blackRedBeLab ```


**Install Node Deps** (in new terminal)

``` npm install ```

**Start Server**

``` npm start ```
